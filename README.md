### 制作アプリのタイトル
myMemoPad - 複数メモ

### 制作アプリの説明（40文字程度）
Local Storageを活用し、複数のメモの作成、保存、編集、削除、一覧表示を可能にしたメモ帳アプリ。

### 工夫した点・こだわった点
1.  **複数メモの管理:** 複数メモの一覧表示と管理を実装しました。
2.  **ハイライト機能:** 選択中のメモがハイライトされるようにし、操作性を向上させました。
3.  **レイアウト:** 編集エリアと一覧エリアを分割するレイアウトにしました。

### 次回トライしたこと（または機能）
1.  **ソート機能:** メモを新しい順に並べ替える機能を追加したい。
2.  **検索機能:** タイトルや本文からキーワードでメモを絞り込めるようにしたい。
